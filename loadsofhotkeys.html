Commands(Windows)-

Windows PowerShell / CMD Commands

  • System Info & Management
  • ipconfig Show IP and network config
    • ipconfig /all Show detailed network info
    • ipconfig /release Release current IP address
    • ipconfig /renew Renew IP address
    • ipconfig /flushdns Flush DNS resolver cache
    • ipconfig /displaydns Display DNS resolver cache
  • ping <address> Check network connectivity
    • ping -t <address> Ping continuously until stopped
    • ping -n <count> <address> Ping with a specific number of requests
  • tracert <address> Trace route to address
  • netstat Show network stats and connections
    • netstat -a Show all active connections and listening ports
    • netstat -n Show addresses and ports numerically
    • netstat -o Show owning process ID for each connection
    • netstat -ano Show all with process IDs
  • File & Disk Commands
  • dir List files & folders
    • dir /p List files one page at a time
    • dir /w List files in wide format
    • dir /a List all files including hidden
  • chkdsk Check disk for errors
    • chkdsk /f Fix errors on the disk
    • chkdsk /r Locate bad sectors and recover readable info
  • System Utilities
  • sfc /scannow Scan and repair system files
  • msconfig Open system configuration
  • regedit Open Registry Editor
  • control Open Control Panel
  • Package & Software Management
  • winget search <app> Search apps with Windows Package Manager
  • winget install <app> Install app via winget
  • winget upgrade Upgrade installed apps

Dev Commands (Git, Node.js, Python)

  • Git Commands
  • git clone <repo-url> Clone a remote repository
  • git status Check current repo status
  • git add <file> Stage changes for commit
  • git commit -m "message" Commit staged changes with message
  • git push Push commits to remote
  • git pull Pull latest changes from remote
  • git branch List branches
    • git branch <name> Create new branch
    • git checkout <branch> Switch branch
    • git merge <branch> Merge branch into current
  • Node.js & npm
  • node --version Check Node.js version
  • npm init Initialize new npm project
  • npm install <package> Install package locally
    • npm install -g <package> Install package globally
    • npm uninstall <package> Remove package
    • npm update Update all packages
  • npm run <script> Run script defined in package.json
  • Python
  • python --version Show Python version
  • python <script.py> Run Python script
  • pip install <package> Install Python package
    • pip uninstall <package> Uninstall package
    • pip list List installed packages
