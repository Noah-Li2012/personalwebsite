How It Works

Transform your videos into terminal art in 4 simple steps

1

Video Input

Start with any video file (MP4, AVI, MOV, etc.)

Works best with MP4
video.mp4
2

Convert to C++

Use Ansimedia to convert video frames into C++ code

ansimedia -in movie.mp4 -out movie.cpp -width 500 -fps 30
This extracts frames, converts to hex colors, and generates optimized C++ code
3

Compile

Compile the generated C++ code into an executable

clang++ movie.cpp -o movie.exe
Optimized for Clang 22.1.0 on Windows
4

Play in PowerShell

Run the executable and watch your video play as ASCII art in the terminal

./movie.exe
The program displays your video frame-by-frame as colored ASCII characters
